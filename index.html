⚙️ One-time setup needed to enable form submissions To allow the form to write to Google Sheets, you need to deploy a free Apps Script endpoint once:

1. Open your Google Sheet → Extensions → Apps Script
2. Paste the script from the APPS_SCRIPT_CODE comment at the bottom of this HTML file
3. Click Deploy → New deployment → Web app → set "Who has access" to Anyone → Deploy
4. Copy the Web App URL and paste it into the APPS_SCRIPT_URL variable in this file

The tracker (reading) works immediately without this step — only submissions need it.
